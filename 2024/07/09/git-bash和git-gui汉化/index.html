git_bash和git_gui汉化

1. git_bash汉化

Git bash本身就支持中文。

只需要在打开Git bash命令窗口后，点击右键Options –>Window –>UI language，下拉选择 zh_CN即可，点击保存。

2. git_gui汉化

git_gui并不能直接设置中文，所以需要手动下载汉化包进行汉化。

本文Git GUI汉化包来源：https://github.com/stayor/git-gui-zh

解压缩后得到 zh_cn.msg文件，然后找到安装git文件的位置。如果默认安装c盘，一般位置应该如下：C:\Program Files\Git\mingw64\share\git-gui\lib

有的版本可能不在program文件下，可以在programx86路径下找找。

进入该路径后，如果有msgs文件夹，直接进入，将文件直接放进去，如果没有，创建msgs文件然后在放进去。

重新打开Git GUI ，就会发现界面已经变成了中文。
