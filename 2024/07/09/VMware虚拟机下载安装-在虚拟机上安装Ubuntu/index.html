VMware虚拟机下载安装+在虚拟机上安装Ubuntu

VMware安装

1. VMware下载

进入官网的 VMware Workstation Pro 页面，根据合适的操作系统选择合适的安装包进行下载。

2. VMware安装

打开下载好的exe文件，开始安装。

安装位置默认在 C 盘下，在这里我选择安装在 D 盘，安装路径尽量不要有中文。可勾选 增强型键盘驱动程序 ，此功能可更好地处理国际键盘和带有额外按键的键盘。

一直下一步，等待软件安装完成。

安装完成后，输入许可证。

将秘钥输入其中

秘钥输入完成后，重启软件，打开就可以正常使用了。

Ubuntu安装

1. 下载 Ubuntu 镜像（iso文件）

  • 官方下载地址：官方下载地址

2. 安装虚拟机

打开 VMware，点击【创建新的虚拟机】。

勾选自定义高级，点击下一步。

  • 选择合适的【Workstation 版本】，然后点击【下一步】：

注：这里的workstation版本选择，对应的是不同的版本支持的不同功能。这里选择的是15.x，安装时根据自己的需求进行选择。

  • 勾选【稍后安装操作系统】，然后点击【下一步】：

客户机操作系统”选择【Linux】，然后“版本”选择【Ubuntu 64 位】，然后点击【下一步】

  • “虚拟机名称”你可以自己起个名字，当然也可以用默认起的名字；然后点击【浏览】选择你要将 Ubuntu 安装的位置（这里建议在系统盘外建立一个新的文件夹，像图中所示的那样）；最后点击【下一步】

“此虚拟机的内存”可以选择默认的 4096 MB，然后点击【下一步】：

勾选【使用网络地址转换(NAT)】，然后点击【下一步】：

  • 选择默认的【LSI Logic（推荐）】，然后点击【下一步】：

  • 选择默认的【SCSI（推荐）】，然后点击【下一步】：

  • 勾选【创建新虚拟磁盘】，然后点击【下一步】：

  • “最大磁盘大小”可以选择 20（需求很大的话可以自行增加，不过 20 还是很够用的，不用担心）；然后勾选【将虚拟磁盘拆分成多个文件】；最后点击【下一步】：

  • 默认即可，点击【下一步】：

  • 选择点击【自定义硬件】：

  • 首先点击【新 CD/DVD 自动检测】，然后勾选【使用 ISO 映像文件】，之后点击【浏览】选择你第一步下载好的镜像文件，最后点击【关闭】：

3. Ubuntu 系统安装

完成上述步骤后，选择刚刚新建的虚拟机

点击开启虚拟机

  • 直接按回车，然后我们要等待一段时间：

一段时间过后，会出现下面的界面。语言我建议英语不好的直接选择【Chinese】，后面使用linux命令不方便再使用命令修改就好。

选择继续

这里建议选择最小安装。正常安装会多出一堆不太用的到的东西比如扫雷等小游戏。

这里选择默认清除整个磁盘并安装Ubuntu，点击现在安装。

点击继续。

根据个人习惯，设置账户，密码。

接下来等待系统安装完成。

安装完成后重启。

重启后，在等待过程中会看到如下界面，点击回车。

完成后，输入刚刚设置的密码进入Ubuntu，页面如下。完成。
